## 曾国藩脚本审视（41分）

### 优点
- **开场强**："你觉得自己不够聪明吗？"直击痛点，小偷嘲笑故事有悬念
- **反差极强**：笨小孩→圣贤，30年坚持日课13条
- **信息密度高**：日课13条具体列出，金句多

### 问题
- 开场"笨功夫"概念有点抽象，可以更具体
- 中间缺少"如果是你"的代入感引导

### 改进建议
```markdown
优化开场：
"你背书要多久？他背了一整夜，被小偷嘲笑太笨。
但30年后，这个笨小孩成了一代圣贤。
他的方法只有一个：笨功夫。"
```

## 范仲淹脚本审视（35分）

- "先忧后乐"金句有价值

- **开场不够强**："你吃过苦吗？"太笼统
- **反差不够具体**：孤儿→名臣，缺少数字冲击

```markdown
"你每天花多少钱吃饭？

他说：这不是省钱，是磨练意志。"
```

## 王阳明脚本审视（39分）

### 优点
- "五溺"概念有趣，有寻找过程
- "龙场悟道"有画面感（石洞、毒虫）
- "心即理"概念有深度

### 问题
- **开场不够直击**："人生低谷"太抽象
- **概念太深**："心即理"普通观众难理解
- **缺少具体行动**：看完不知道怎么做

### 改进建议
```markdown
优化开场：
"你遇到过人生最低谷吗？
他37岁被贬到荒蛮之地，住石洞，采野菜，差点病死。
但正是这里，他悟出了影响中国500年的思想。"

简化概念：
把"心即理"改成"答案在你心里"
把"致良知"改成"你的心本就知道善恶"
```

## 苏东坡脚本审视（42分）⭐最强

- **反差极强**：黄州→惠州→儋州，把逆境当功业

- 中间可以加"如果是你会怎样"

```markdown
优化开场：
"你被降职过吗？他一生被贬三次，越贬越远，最后到了天涯海角。
但他却说：问汝平生功业，黄州惠州儋州。
他把逆境，当成了功业。"
```


## 爆款优化原则总结

### 1. 开场必须有痛点共鸣
- 不要问抽象问题："你遇到过XX吗？"
- 要问具体问题："你每天花多少钱吃饭？"
- 要有数字冲击："他坚持了3年"

### 2. 反差必须有数字
- 不要说："从低谷到高峰"
- 要说："笨小孩 → 圣贤（30年）"
- 要说："三次被贬 → 越活越洒脱"

### 3. 概念必须简化
- 不要说："心即理"
- 要说："答案在你心里"
- 不要说："致良知"
- 要说："你的心本就知道善恶"

### 4. 必须有代入感引导
- 不要只讲故事
- 要问："如果是你"
- 要说："你也可以"

### 5. 结尾必须有行动号召
- 不要只说下期预告
- 要说："点赞关注，明天见"
- 要说："记住这个道理"
